## 1. 개발 목표
* 비주얼 slider
  * JQuery로 구현한 코드
  * Circle-Progress 플러그인 사용하여 인터렉티브한 효과 구현
* 시맨틱 태그 / Flex 레이아웃 / 반응형
* 웹접근성 고려한 focus 이벤트 연결

 
 

  
  
   
## 2. 작업 정보
* 하드코딩 / 개인프로젝트 / 반응형
* 기간 : 2주 / 담당 : 퍼블리싱 100% 
* 사용기술 : HTML, CSS, JQuery
* 개발환경 : Visual Studio Code, git, github
* JQuery UI 제작
  * GNB 
  * Visual Slider
  * TAB Slider
  * Toggle mene (전체메뉴)
  * Top 이동 버튼
* JQuery Plugin
  * Circle-Progress
