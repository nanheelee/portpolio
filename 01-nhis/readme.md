# 건강보험공단

## 1. 개발 목표
* div태그 레이아웃 → 시맨틱 태그 구조 변경
* Float 레이아웃 → Flex 레이아웃 변경
* 웹표준 준수
* 여러 개의 TAB메뉴 함수로 동작 
* 미디어쿼리 이용한 반응형 구현
* 상단 GNB와 전체메뉴(Full Menu) - 반응형 구현
