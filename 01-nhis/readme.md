# 건강보험공단 (clon coning)

사이트 데모 : [<http://sachory.dothome.co.kr/>](http://sachory.dothome.co.kr/01-nhis/index.html)

## 1. 개발 목표
* div태그 레이아웃 → 시맨틱 태그 구조 변경
* Float 레이아웃 → Flex 레이아웃 변경
* 상단 GNB와 전체메뉴(Full Menu) - 반응형 구현
* 여러 개의 TAB메뉴 - 함수로 동작 
* 웹접근성 고려한 focus 이벤트 연결
* 웹표준 준수
* 미디어쿼리 이용한 반응형 구현
