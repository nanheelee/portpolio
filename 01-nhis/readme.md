# 건강보험공단

## 1. 개발 목표
* div태그 레이아웃 → 시맨틱 태그 구조 변경
* Float 레이아웃 → Flex 레이아웃 변경
* 웹표준 준수
* 여러 개의 TAB메뉴 함수로 동작 
* 미디어쿼리 이용한 반응형 구현
* 상단 GNB와 전체메뉴(Full Menu) - 반응형 구현
* 웹접근성 고려한 focus 이벤트 연결



## 2. 작업 정보
* 하드코딩 / 개인프로젝트 / 클론코딩
* 기간 : 2주 / 담당 : 퍼블리싱 100%
* 사용기술 : HTML, CSS, JQuery, 
* 개발환경 : Visual Studio Code, git, github
