## 1. 개발 목표
* 시맨틱 태그
* Flex 레이아웃
* 상단 GNB와 전체메뉴(Full Menu) - 반응형 구현
* 여러 개의 TAB메뉴 - 함수로 동작 
* 웹접근성 고려한 focus 이벤트 연결
* 웹표준 준수
* 미디어쿼리 이용한 반응형 구현
